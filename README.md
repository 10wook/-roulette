# 🎰 룰렛 게임 웹사이트

룰렛 추첨 게임을 위한 웹 애플리케이션입니다. 참여자 정보를 입력하고 룰렛을 돌려 추첨 결과를 확인할 수 있으며, 참여자 관리 및 데이터 내보내기 기능을 제공합니다.

## ✨ 주요 기능

### 1. 참여자 등록
- 이름과 전화번호 뒷자리 4자리로 참여자 등록
- 중복 참여 방지 (이름 + 전화번호 조합으로 체크)
- 입력 검증 (이름 필수, 전화번호는 숫자 4자리만)

### 2. 룰렛 게임
- 8개 섹션으로 구성된 룰렛
- 부드러운 회전 애니메이션 (3~5바퀴 랜덤 회전)
- 실시간 결과 표시
- 포인터가 가리키는 숫자와 결과 일치

### 3. 데이터 관리
- 참여자 정보 및 추첨 결과 자동 저장
- 브라우저 로컬스토리지에 데이터 저장
- 참여자 목록 확인 및 수정/삭제 기능
- JSON 및 엑셀 파일로 데이터 내보내기

### 4. 관리자 패널
- 저장된 모든 참여자 정보 확인
- 참여자 정보 수정 (이름, 전화번호, 추첨 결과)
- 개별 참여자 삭제
- 전체 데이터 삭제
- 데이터 내보내기 (JSON, 엑셀)

## 🛠 기술 스택

- **HTML5**: 웹 페이지 구조
- **CSS3**: 스타일링 및 애니메이션
- **JavaScript (Vanilla)**: 게임 로직 및 데이터 관리
- **SheetJS (xlsx)**: 엑셀 파일 생성 라이브러리
- **LocalStorage**: 브라우저 로컬 데이터 저장

## 🚀 Vercel 배포

이 프로젝트는 Vercel에 바로 배포할 수 있습니다. 정적 웹사이트이므로 추가 설정 없이 배포 가능합니다.

### 배포 방법

#### 방법 1: Vercel CLI 사용

1. **Vercel CLI 설치**
   ```bash
   npm i -g vercel
   ```

2. **프로젝트 디렉토리에서 배포**
   ```bash
   cd -roulette
   vercel
   ```

3. **배포 확인**
   - CLI에서 제공하는 URL로 접속하여 확인

#### 방법 2: Vercel 웹 대시보드 사용

1. **GitHub에 프로젝트 업로드** (선택사항)
   - GitHub 저장소 생성
   - 프로젝트 파일 업로드

2. **Vercel 대시보드 접속**
   - [vercel.com](https://vercel.com) 접속
   - GitHub 계정으로 로그인

3. **새 프로젝트 추가**
   - "Add New Project" 클릭
   - GitHub 저장소 선택 또는 파일 직접 업로드

4. **배포 설정**
   - Framework Preset: "Other" 또는 "Static Site"
   - Build Command: 없음 (또는 비워둠)
   - Output Directory: `.` (현재 디렉토리)

5. **배포 실행**
   - "Deploy" 버튼 클릭
   - 배포 완료 후 제공되는 URL로 접속

#### 방법 3: 드래그 앤 드롭

1. **Vercel 대시보드 접속**
   - [vercel.com](https://vercel.com) 접속

2. **프로젝트 폴더 업로드**
   - 프로젝트 폴더를 드래그 앤 드롭
   - 자동으로 배포됨

### 배포 후 확인사항

- ✅ 웹사이트가 정상적으로 로드되는지 확인
- ✅ 참여자 등록 기능 테스트
- ✅ 룰렛 게임 동작 확인
- ✅ 관리자 패널 접근 확인
- ✅ 데이터 저장 기능 확인 (로컬스토리지)

### 주의사항

- **데이터 저장**: 로컬스토리지는 브라우저별로 저장되므로, 사용자마다 독립적인 데이터를 가집니다.
- **HTTPS**: Vercel은 자동으로 HTTPS를 제공합니다.
- **도메인**: Vercel은 무료로 커스텀 도메인을 연결할 수 있습니다.

### 배포된 사이트 예시

배포가 완료되면 다음과 같은 URL이 제공됩니다:
- `https://your-project-name.vercel.app`

## 📖 사용 방법

### 참여자 등록 및 게임 플레이

1. **참여자 정보 입력**
   - 이름 입력
   - 전화번호 뒷자리 4자리 입력
   - "참여하기" 버튼 클릭

2. **룰렛 돌리기**
   - 룰렛 게임 화면이 표시됨
   - "돌리기" 버튼 클릭
   - 룰렛이 회전하며 결과 표시

3. **결과 확인**
   - 추첨 결과 숫자 표시
   - 결과는 자동으로 저장됨
   - "처음으로 돌아가기" 버튼으로 메인 화면으로 이동

### 관리자 기능 사용

1. **관리자 패널 열기**
   - 메인 화면 상단의 "관리자" 버튼 클릭

2. **참여자 목록 확인**
   - 저장된 모든 참여자 정보 확인
   - 이름, 전화번호, 참여일시, 추첨 결과 표시

3. **데이터 수정**
   - 참여자 항목의 "수정" 버튼 클릭
   - 이름, 전화번호, 추첨 결과 수정 가능
   - "저장" 버튼으로 변경사항 저장

4. **데이터 삭제**
   - 개별 삭제: 각 참여자의 "삭제" 버튼 클릭
   - 전체 삭제: "전체 삭제" 버튼 클릭 (확인 필요)

5. **데이터 내보내기**
   - **JSON 내보내기**: JSON 형식으로 데이터 다운로드
   - **엑셀 내보내기**: 엑셀 파일(.xlsx)로 데이터 다운로드
     - 엑셀 파일에는 번호, 이름, 전화번호, 참여일시, 추첨결과, 결과일시가 포함됨

## 💾 데이터 저장

### 저장 위치
- **브라우저 로컬스토리지 (LocalStorage)**
- 키 이름: `roulette_participants`
- 브라우저별로 독립적으로 저장됨

### 저장되는 데이터 형식

```json
[
  {
    "name": "홍길동",
    "phone": "1234",
    "date": "2025-01-15T10:30:00.000Z",
    "result": 5,
    "resultDate": "2025-01-15T10:35:00.000Z"
  }
]
```

### 데이터 특징
- 브라우저를 닫아도 데이터 유지
- 브라우저 캐시/데이터 삭제 시 사라짐
- 다른 브라우저나 기기에서는 접근 불가
- 시크릿 모드에서는 별도 저장소 사용

### 데이터 백업
- 관리자 패널에서 JSON 또는 엑셀 파일로 내보내기 가능
- 정기적으로 데이터를 백업하는 것을 권장

## 🎮 룰렛 게임 상세

### 룰렛 구성
- **섹션 수**: 8개 (1~8번)
- **각 섹션 각도**: 45도씩
- **회전 방향**: 시계 반대 방향
- **회전 바퀴 수**: 3~5바퀴 (랜덤)

### 결과 계산
- 포인터(위쪽)가 가리키는 섹션의 숫자가 결과
- 회전 각도 계산을 통해 정확한 결과 표시
- UI에 표시된 숫자와 결과 숫자 일치

## 🔧 주요 함수 설명

### 참여자 관리
- `getParticipants()`: 로컬스토리지에서 참여자 목록 가져오기
- `saveParticipant(name, phone)`: 참여자 정보 저장
- `saveParticipantResult(name, phone, resultNumber)`: 추첨 결과 저장
- `isDuplicateParticipant(name, phone)`: 중복 참여 체크

### 룰렛 게임
- `spinRoulette()`: 룰렛 회전 함수
- `getCurrentRotation()`: 현재 룰렛 회전 각도 가져오기

### 관리자 기능
- `displayParticipants()`: 참여자 목록 표시
- `editParticipant(index)`: 참여자 정보 수정 모드
- `deleteParticipant(index)`: 참여자 삭제
- `exportParticipantsJSON()`: JSON 파일로 내보내기
- `exportParticipantsExcel()`: 엑셀 파일로 내보내기

## ⚠️ 주의사항

1. **데이터 백업**: 브라우저 데이터 삭제 시 모든 정보가 사라지므로 정기적으로 백업 권장
2. **중복 참여**: 같은 이름과 전화번호 조합으로는 한 번만 참여 가능
3. **브라우저 호환성**: 최신 브라우저 사용 권장
4. **인터넷 연결**: SheetJS 라이브러리 로드를 위해 초기 로딩 시 인터넷 연결 필요
